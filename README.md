# TextImageMaker

将文字制作成图片

将输入的文字制作成图片，以此方便用于在朋友圈动态的内容上发布时不再为没有图片而烦恼。

#### 功能：
1. 可以选择图片背景;
2. 可以指定文字的风格：颜色等;
3. 可以一键分享到社交平台，如朋友圈;
4. 自定义文本位置，居中，居左和居右
